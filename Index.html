In this tutorial, you'll learn how to create a basic HTML table.

    </tr>

        <td>Mandrake Root</td>

        <td>Rare</td>
    </tr>
    <tr>
        <td>Phoenix Feather</td>
        <td>Magical</td>
        <td>Very Rare</td>
    </tr>
</table>
